## Installation

This project and its dependencies:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Sanzaya8441/Cine-Hub-app](https://github.com/Sanzaya8441/Cine-Hub-app)

2. **Navigate to the project directory:**
   ```bash
   cd Cine-Hub-app

3. **Install dependencies: Using pub:**
    ```bash
    flutter pub get

4. **Run the application: Using dart:**
   ```bash
   dart run ./Cine-Hub-app
